# VibeKE 🎬🇰🇪

> **Your Mood. Your Vibe. Your Kenya.**

VibeKE is a Kenyan entertainment discovery platform where users pick their mood and get a movie or anime recommendation. They can also browse Kenyan movies, creators and artists, add their own recommendations, and subscribe to the newsletter.

---

## Pages

| File       | Page | Purpose                                                |
| ---------- | ---- | ------------------------------------------------------ |
| index.html | Home | Mood recommender + community picks + newsletter signup |
| hub.html   | Hub  | Browse Kenyan entertainment + search and filter        |
| add.html   | Add  | Submit a recommendation + view past submissions        |

---

## Features

- Mood recommender — pick your mood, get a recommendation instantly
- Kenyan entertainment hub — movies, creators and artists
- Search and filter — find content by keyword or category
- Add recommendations — saved to localStorage and shown on the hub
- Newsletter signup — name and email saved to localStorage
- Cross-page data persistence — data added on one page shows on another

---

## Tech Stack

- HTML5 — semantic structure
- CSS3 — deep purple theme, CSS Grid, Flexbox, media queries
- Vanilla JavaScript — DOM manipulation, event handling, localStorage

---

## How to Run

1. Clone the repo:

```bash
git clone git@github.com:johnchriswamai/vibe-ke.git
```

2. Open `index.html` in your browser
3. No installs or server needed
